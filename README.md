# UI-library

Данный проект представляет собой тестовую библиотеку ui-компонентов для фронтенда на react.

После внесения изменений, необходимо выполнить следующие команды, чтобы обновить версию в packages </br>
1. ```npm run build``` для проверки, что сборка проходит без ошибок </br>
2. ```npm version patch``` для обновления версии в package.json </br>
3. ```npm publish``` для публикации новой версии </br>
