#UI-library

Данный проект представляет собой тестовую библиотеку ui-компонентов для фронтенда на react.

После внесения изменений, необходимо выполнить следующие команды, чтобы обновить версию в packages
```npm run build``` для проверки, что сборка проходит без ошибок
```npm version patch``` для обновления версии в package.json
```npm publish``` для публикации новой версии
